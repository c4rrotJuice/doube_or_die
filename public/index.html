How to Play

  1. Tap START + DOUBLE to begin a run and double your multiplier.
  2. Each double increases both reward and crash risk.
  3. Use CASH OUT before a crash to bank your multiplier.
  4. If you crash, the run ends with no banked gain.

Keyboard: D = Double, C = Cash Out, H = Open help, Esc = Close modal.
